# Kode utsending

En feil i koden førte til at kode utsending ikke virket for nye systemer. Denne feilen er nå rettet opp.

# Epost utsending

Post vil nå sendes fra [hundehall.no](http://hundehall.no) og via epost relay tjenesten til mailbaby. Overgangen kommer til å gå gradvis for å sikre at dette fungerer som tenkt.

# Booking tilbake i tid

Booking har til nå vært åpen for alle timer med dagens dato og fremover. Dette førte til at man kunne booke timer som hadde passert for mange timer siden. 

Denne versjonen innskrenker dette til at man kun kan booke inneværende time og fremover. Ved behov vil dette bli en innstilling som man kan justere på som systemeier.
