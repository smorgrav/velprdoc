## Hva er regler?

Regler i dette systemet er tekst som brukere må(\*) godta før de kan gjøre en booking. Reglene kan være generelle regler for bruk av tjenesten, regler for en spesifikk enhet, eller annen informasjon som er viktig for brukeren å vite før de booker.


<Aside>Man kan styre om man vil ha en aksept for reglene en gang, alle gangene eller ved forandring av reglene.</Aside>

## Generelle regler

Regler kan brukes til å kommunisere og aktivt innta en aksept for en bestemt type adferd. Hunden skal være luftet, det er ikke lov til å lufte hunden på naboens område eller lignende.
Kjært barn ha mange navn som hallreker, parkregler og trivselsregler.

## Regler som ekstra utleie informasjon

Ett vanlig bruksområde for regler er å vise informasjon om en utleie enhet ved booking. Dette kan være informasjon om hvordan utstyret skal brukes, hvilke regler som gjelder, eller annen informasjon som er viktig for brukeren å vite før de booker.

## Regler som betalingsinformasjon

For de som ikke har mulighet til å sette opp direkte betaling via systemet kan regler brukes til å informere om betalingsinformasjon. Dette kan være kontonummer, Vippsnummer, eller annen informasjon som er nødvendig for å gjennomføre betaling.
